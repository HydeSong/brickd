# 组件

brickd 提供了丰富的 React 组件，满足您的各种业务需求。

## 组件分类

### 基础组件

- Button - 按钮
- Icon - 图标
- Typography - 排版

### 布局组件

- Layout - 布局
- Grid - 网格
- Flex - 弹性布局
- Space - 间距

### 导航组件

- Menu - 菜单
- Tabs - 标签页
- Breadcrumb - 面包屑
- Pagination - 分页
- Steps - 步骤条
- Anchor - 锚点

### 表单组件

- Form - 表单
- Input - 输入框
- Select - 选择器
- Checkbox - 复选框
- Radio - 单选框
- Switch - 开关
- DatePicker - 日期选择器
- TimePicker - 时间选择器
- InputNumber - 数字输入框
- Cascader - 级联选择器
- Rate - 评分
- Slider - 滑块
- ColorPicker - 颜色选择器
- Transfer - 穿梭框

### 反馈组件

- Modal - 对话框
- Drawer - 抽屉
- Message - 消息提示
- Notification - 通知
- Popover - 气泡卡片
- Tooltip - 提示
- Alert - 警告
- Progress - 进度条

### 数据展示

- Table - 表格
- Card - 卡片
- List - 列表
- Avatar - 头像
- Badge - 徽标
- Calendar - 日历
- Carousel - 轮播
- Collapse - 折叠面板
- Descriptions - 描述列表
- Empty - 空状态
- Image - 图片
- Masonry - 瀑布流
- QRCode - 二维码
- Segmented - 分段控制器
- Statistic - 统计数值
- Tag - 标签
- Timeline - 时间轴
- Tree - 树形控件
- TreeSelect - 树形选择器

### 其他组件

- Affix - 固钉
- AutoComplete - 自动完成
- Breadcrumb - 面包屑
- ConfigProvider - 配置提供者
- Divider - 分隔线
- Dropdown - 下拉菜单
- FloatButton - 悬浮按钮
- Mentions - 提及
- Popconfirm - 气泡确认框
- Skeleton - 骨架屏
- Spin - 加载中
- Splitter - 分割线
- Tour - 引导
- Upload - 上传
- Watermark - 水印
